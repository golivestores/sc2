# sc2 effect 标签规范

每个 `effects/NNN-name/meta.json` 的 `tags` 字段必须按下面的**五轴分类**填，由 `package-effects.py` 校验。违反就 fail-fast，新 effect 进不了画廊。

---

## 规则速查

| 轴 | 必填？ | 数量 | 用途 |
|---|---|---|---|
| 1 板块 | ✅ 必填 | **恰好 1 个** | 网站语义位置（用在哪里）|
| 2 形态 | ⭕ 可选 | 0 或 1 | UI 视觉形状 |
| 3 触发 | ✅ 必填 | **恰好 1 个** | 怎么激活动效 |
| 4 技术 | ⭕ 可选 | 0–3 | 用什么技法做的 |
| 5 产品类型 | ✅ 必填 | **恰好 1 个** | 源站所属行业 |

`tags` 数组里把这些值平铺即可，顺序无所谓。

```json
"tags": ["产品", "网格", "悬停", "路径裁切", "美妆"]
```

---

## 轴 1：板块（必填 1）

哪个网站板块用这个效果。

| tag | 含义 |
|---|---|
| `首屏` | 顶部 hero / landing 主视觉 |
| `导航` | 顶 bar、header、菜单系统 |
| `产品` | 商品列表 / 产品卡片 |
| `分类` | 品类导航 / collection grid |
| `服务` | 业务能力 / features / capabilities |
| `流程` | 操作步骤 / how-it-works / 原理介绍 |
| `评价` | 用户评价 / testimonials |
| `新闻` | 新闻 / 博客 / 文章列表 |
| `案例` | 作品集 / portfolio / 项目展示 |
| `CTA` | 行动召唤区块 |
| `关于` | 公司/品牌介绍段 |
| `联系` | 联系表单 / 地图 / 客服入口 |
| `页脚` | footer |

---

## 轴 2：形态（可选 0–1）

适用于有明确视觉模式的；纯文字段如 hero/CTA 留空。

| tag | 含义 |
|---|---|
| `网格` | 卡片网格阵列 |
| `轮播` | 多帧切换（明确的 N 张图） |
| `滑块` | 单帧 + 缩放/morph |
| `跑马灯` | 跑马灯文字带 |
| `顶栏` | 横向粘滞 bar |
| `横幅` | 大图 banner |
| `文字段` | sticky / pinned 文字段 |
| `列表` | 纵向列表项 |

---

## 轴 3：触发（必填 1）

| tag | 含义 |
|---|---|
| `入场` | 加载时一次性 |
| `滚动` | 滚动位置驱动（含 sticky pinning）|
| `悬停` | mouseenter |
| `点击` | tap/click 切态 |
| `自动播放` | 无需用户介入，自动循环 |

---

## 轴 4：技术（可选 0–3）

| tag | 含义 |
|---|---|
| `形变` | 形态/尺寸/位置插值过渡 |
| `路径裁切` | clip-path 揭示或形变 |
| `逐字出场` | 文字按字/词错峰出场 |
| `视差` | parallax 视差/景深 |
| `交叉淡入` | 多元素 opacity 互切 |
| `多态切换` | ≥3 个离散状态 |
| `遮罩` | 渐变或形状 mask |
| `文字滚动` | marquee-style 文字内部循环 |
| `响应式` | 显式做了 mobile/desktop 差异化 |
| `键盘` | 支持键盘控制 |

---

## 轴 5：产品类型（必填 1）

源站所属行业。新行业进来如果都不匹配，先提个 PR 加进表里再用。

| tag | 含义 |
|---|---|
| `美妆` | 化妆品 / skincare |
| `食品` | 食品饮料 / charcuterie / F&B |
| `健康` | 医药 / wellness / 保健 |
| `时尚` | 服装 / 配饰 |
| `B2B服务` | 企业服务 / agency / SaaS |
| `工业安防` | industrial / defense / cybersecurity |
| `协会组织` | 行业协会 / NGO |
| `设计建筑` | 设计工作室 / 建筑事务所 |
| `科技` | AI / 数字产品 |
| `教育` | — |
| `金融` | — |
| `房产` | — |
| `生活方式` | 家居 / hobby / lifestyle |

## 示例

**美妆产品 grid + hover 揭示**：
```json
"tags": ["产品", "网格", "悬停", "路径裁切", "美妆"]
```

**B2B 首屏 + 入场 + 文字逐字出场**（无明确形态）：
```json
"tags": ["首屏", "入场", "逐字出场", "B2B服务"]
```

**食品导航条 + 滚动收缩 + 多态切换 + 响应式**：
```json
"tags": ["导航", "顶栏", "滚动", "多态切换", "形变", "响应式", "食品"]
```

---

## 不合规会怎样

`package-effects.py` 启动时校验所有 `meta.json` 的 tags，发现下列问题之一就 abort：

- 缺板块、缺触发、缺产品类型
- 出现表外的 tag（拼写错误、用了 deprecated 词）
- 形态 ≥ 2 个、技术 ≥ 4 个

错误信息会告诉你具体哪个 effect 的哪个轴有问题，去修了再 retry。
